# GTA Third-Person Pack

Пакет под `GTA`-стилистику для `ComfyUI`, собранный вокруг сценария:

- персонаж в кадре со спины
- камера следует сзади или чуть через плечо
- игровой cinematic vibe
- короткие вертикальные ролики под `Reels` / `TikTok`

## Что внутри

- `LORAS_AND_TRIGGERS.md`
  Подобранные LoRA, триггеры и рабочие веса.
- `NODES.md`
  Какие ноды нужны для workflow.
- `PROMPTS.md`
  Готовые промпты под ключкадр и видео.
- `PIPELINES.md`
  Какие пайплайны использовать и в каком порядке.
- `models/`
  Файлы-заглушки вместо больших моделей и LoRA с ссылками, именами файлов и путями для `ComfyUI`.
- `workflows/`
  Подготовленные workflow-файлы и копии curated-шаблонов.

## Workflow-файлы в пакете

- `workflows/gta_flux_klein_keyframe.json`
  Template для стартового keyframe / restyle-кадра в GTA-направлении.
- `workflows/gta_wan_third_person_animator_5090_safe.json`
  Основной видео-workflow для движения камеры сзади и короткого GTA-like ролика.
- `workflows/gta_video_ultrasharp_1080p.json`
  Финальный upscale до `1080p`.

## Рекомендуемая схема

1. Делаем ключкадр в `FLUX`.
2. Добавляем стиль через `GTA` LoRA.
3. Для ракурса используем промпт `rear view / third person / over the shoulder`.
4. Анимируем в видео через `Wan Animate`.
5. Финально прогоняем upscale.

## Важная оговорка по совместимости

Главная `GTA VI Style` LoRA из этого пакета обучена под `FLUX.1 D`.

- Лучший результат по стилю будет на `FLUX.1-dev` или совместимой `FLUX.1` базе.
- В репозитории также добавлен template на базе `Flux.2 Klein`, но его нужно воспринимать как удобную стартовую заготовку.
- Если стиль слабо цепляется на `Flux.2 Klein`, лучше перейти на `FLUX.1-dev` и оставить те же промпты и LoRA.

## Быстрый старт

1. Открой `models/` и скачай недостающие веса по ссылкам.
2. Поставь ноды из `NODES.md`.
3. Возьми промпт из `PROMPTS.md`.
4. Открой workflow из `workflows/`.
5. Подставь свой стартовый кадр или исходное видео.

## Что здесь не хранится

- большие модели
- бинарные LoRA
- кастомные ноды как исходники

Вместо этого лежат безопасные для публичного репозитория текстовые заглушки с источниками загрузки.
